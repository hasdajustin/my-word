# My Word

**My Word** is a full-stack blog web application built with **Django REST Framework** and **React (Vite + Tailwind CSS)**.  
Users can read blog posts, while the author can create, update, and delete posts with session-based view tracking.

---

## 🚀 Features

-  Read blog posts  
-  Session-based view count  
-  Create, edit, delete posts (Author only)  
-  Image upload support  
-  SEO-friendly slug URLs  
-  Dashboard with total posts & views  
-  Fast frontend with Vite + React  
-  Modern UI with Tailwind CSS  

---

## 🛠️ Tech Stack

### 🔙 Backend
- Django  
- Django REST Framework  
- PostgreSQL  

### 🔜 Frontend
- React (Vite)  
- Tailwind CSS  
- Axios  

---

## 🌐 API Endpoints

| Method | Endpoint | Description |
|--------|---------|-------------|
| GET    | /api/posts/ | Get all posts |
| GET    | /api/posts/&lt;slug&gt;/ | Get single post by slug |
| POST   | /api/posts/ | Create a new post |
| PUT    | /api/posts/&lt;slug&gt;/ | Update a post by slug |
| DELETE | /api/posts/&lt;slug&gt;/ | Delete a post by slug |
